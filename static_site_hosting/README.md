# 静的サイトホスティング用テンプレート
## リソース
- S3 Bucket (1)
  - 公開状態
  - バケットポリシーでアクセス制御する
- S3 Bucket Policy (2)
  - (1)のバケットポリシー
  - パブリック読み取りを全て許可する
- Cloud Front (3)
  - (1)をオリジンにしてコンテンツを配信する
  - ACMで作成済みSSL証明書を利用する（SNIをサポート）
  - 全パスパターンで同じキャッシュ動作
- Route53のAレコード (4)
  - (3)を配信先として指定する
  - Cloud Frontで配信するので、固定のHostedZoneIdを指定している
